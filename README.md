# Frontend-Practice-Collection

A frontend practice sandbox featuring CSS layout experiments, Bootstrap grid demos, and interactive component examples.

## ✅ What’s Included

- `index.html`: Main CSS mastery page with examples for sticky header, input states, iframe embed, table layout, margin collapse, relative positioning, and transform demos.
- `BootstrapGrid.html`: Animated Bootstrap 5 grid playground with nested column layouts and hover interactions.
- `2d3d.html`: Transform demo page for 2D/3D interactions.
- `style/style.css`: Core styles for `index.html`.
- `style/bootstrapGrid.css`: Animated dark visual style for Bootstrap grid page.

## 🚀 Features Added

- Gradient dark UI and visual glow effects
- Card hover lift and interactive transitions
- Sequential row entrance animation for better visualization
- Responsive layout behavior for tablet/mobile screens
- Navigation CTA from `index.html` → `BootstrapGrid.html`
- Return button from `BootstrapGrid.html` → `index.html`

## ▶️ How to Run

1. Open `index.html` in your browser.
2. Click the “Open Bootstrap Grid Demo” button at the bottom.
3. Use the return button in `BootstrapGrid.html` to go back to home.

## 🧠 Learnings

This project demonstrates:
- Bootstrap 5 grid basics (`.row`, `.col-` sizing)
- Nested Bootstrap columns and responsive breakpoints
- CSS animations (`@keyframes`, `transition`, hover effects)
- Visual composition with gradients, shadows, and glassy surfaces
